SimpleNews

SimpleNews es un proyecto que busca crear un código base abierto para la gestión de noticias.

El objetivo es crear una plataforma sencilla y fácil de usar para la publicación y gestión de noticias.

Esta versión básica permite la creación de noticias, la gestión de usuarios y la búsqueda de noticias; así como comentar.
